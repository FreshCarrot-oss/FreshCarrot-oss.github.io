# 🧠 PsychoTest

> Научная платформа психологического тестирования с AI-анализом и glassmorphism-дизайном

## 🚀 Быстрый старт

```bash
# 1. Клонировать репозиторий
git clone https://github.com/username/psychotest.git && cd psychotest

# 2. (Опционально) Настроить Firebase
python scripts/setup_firebase.py

# 3. Запустить локальный сервер
python scripts/run_local.py
```

Откроется браузер с адресом `http://localhost:8080`

## 🤖 AI-анализ

Платформа генерирует структурированный промпт на основе результатов всех пройденных тестов. Вы копируете его и вставляете в ChatGPT, Claude или Gemini в своём аккаунте. **Никаких API-ключей, никаких данных на сторонних серверах.**

## 🛠 Стек технологий

- **Frontend:** HTML5, CSS3, JavaScript ES6+ (модули)
- **Анимации:** GSAP 3, tsParticles, CSS @keyframes
- **Графики:** Chart.js 4
- **База данных:** Firebase Firestore
- **Авторизация:** Firebase Auth (email/пароль + Google)
- **Хостинг:** GitHub Pages
