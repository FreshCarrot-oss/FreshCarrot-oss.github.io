# FreshCarrot-oss.github.io

# 🌌 Prokofiev verse — Concept Website

Небольшой концептуальный сайт, посвящённый вселенным Романа Прокофьева:  
**Stellar**, **Звёздная Кровь** и **Игра Кота**.

🔗 Демо: https://freshcarrot-oss.github.io/

---

## 📌 О проекте

Это не полноценный продукт, а **дизайнерская заглушка**, созданная для:
- экспериментов с интерфейсом  
- проработки атмосферы  
- теста визуальных решений  

Сайт больше про **ощущения и стиль**, чем про функциональность.

---

## ⚙️ Функционал

На данный момент:

- 🎨 Проработанный визуальный дизайн  
- 🧭 Навигация и интерактивные элементы  
- 🧠 Прототип "когитора" (нейросети внутри сайта)  
- 🖥️ Панель **Гранд-Координатора**  

❗ Важно:  
Нейросеть работает как **концепт**, без реального API.

---

## 🧩 Панель Гранд-Координатора

В сайте есть скрытая админ-панель:

**Как открыть:**
1. Нажать `System` в правом верхнем углу  
2. Найти пароль прямо на сайте 👀
